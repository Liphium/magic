# Liphium Magic: Docker builds for everyone.

While the goal of this project is not entirely clear, we want to make it incredibly easy, fast and cheap to build Docker images in the Cloud. There is no reason for GitHub Actions or any other CI/CD provider to charge you cents by the minute. Liphium Magic is supposed to give you this experience for cheap with a nice GitHub integration that can build Docker images for every branch or pull request.

Furthermore, it's also supposed to be a foundation for other ideas that we come up with during development. The end goal of this project is to make the Cloud fun and cheap, with the option of self-hosting everything with ease.
